                       # Openvino basic start and install information

A. Install application

1. Register and download Linux installer from https://software.intel.com/en-us/openvino-toolkit/choose-download/free-download-linux
2. Copy to .sh file and .tgz file to a specific folder
3. sudo ./openvino_setup.sh
4. Follow up instructions shown in order for complete install
